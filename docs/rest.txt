<h2>REST (Representational State Transfer)</h2>

<p>
    Modelo de comunicacion que se utiliza para el intercambio de datos entre Cliente y servidor, utiliza los  siguientes metodos HTTP (HyperText Transfer Protocol).
</p>

<ul>    
    <LI>GET: solicita un recurso al servidor</LI>
    <LI>POST: enviar datos al servidor</LI>
    <LI>PUT: solicita la modificacion de datos</LI>
    <LI>PATCH: envia la modificacion de un valor</LI>
    <LI>DELETE: solicita la eliminacion de datos</LI>
</ul>
