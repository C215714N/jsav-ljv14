<h2>AJAX (Asynchronous Javascript And XML)</h2>

<p>
    Estructura de comunicacion que define como debe realizarse la comunicacion entre Cliente y Servidor, surge para facilitar la comunicacion entre FrontEnd y Backend. Se incorpora informacion de las cabeceras y se procesa utilizando una instancia de la clase XMLHttpRequest(), que mediante Callback retorna los valores resultantes de la peticion que se traducen como:
</p>

<ul>
    <li>0: Instancia no incializada</li>
    <li>1: Objeto incializado</li>
    <li>2: En proceso de envio</li>
    <li>3: Descarga de respuesta</li>
    <li>4: Comunicacion Finalizada</li>
</ul>
